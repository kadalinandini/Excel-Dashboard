# Excel-Dashboard
Excel-based Cold Coffee Sales Dashboard that analyzes sales data, revenue trends, product performance, and key business metrics using charts, pivot tables, and data visualization techniques.

**Project Overview**
  This project presents an interactive Coffee Sales Dashboard built in Microsoft Excel to analyze sales performance between 2019 and 2021.
The dashboard uses Pivot Tables, slicers, charts, and data formatting to transform raw sales data into meaningful business insights.
It helps users easily explore sales trends, customer behavior, and regional performance through interactive filters.

**Dashboard Filters**
The dashboard includes interactive filters to analyze data based on:
Order Date (Month & Year)
Roast Type (Dark, Medium, Light)
Coffee Size (0.2kg, 0.5kg, 1.0kg, 2.5kg)
Loyalty Card Status (Yes / No)

**Key Dashboard Insights**
The dashboard provides several insights including:
Monthly Sales Trends from 2019–2021.
Sales by Country showing regional performance.
Top 5 Customers based on total purchases.
Product preferences based on roast type and size.
Impact of loyalty card usage on sales.
